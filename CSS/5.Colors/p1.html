colors

        1. color name
        2. hexa decimal (6bits) solid color, (last 2bits) -> transparency
        3. rgb(0,0,0) - rgb(255,255,255)
        4. cyan,yellow,megenta (secondary colors)
        5. rgba(red,green,blue,alpha) = alpha(transparency) 0.1 - 1.0
        6. hsl (hue,saturation,lightness)
            hsl(0-255,0-100%,0-100%)
        7. hsla(hue,saturation,lightness,alpha)
        alpha(transparency) 0.1 - 1.0
